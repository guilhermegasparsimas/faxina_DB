🧹 Faxina_DB
Sistema completo de agendamento de serviços de limpeza, desenvolvido com arquitetura moderna, integrando front-end, back-end e banco de dados de forma eficiente e escalável.

A aplicação permite o cadastro e autenticação de usuários, além do gerenciamento de agendamentos de faxinas, oferecendo uma experiência simples, rápida e organizada tanto para clientes quanto para administração do serviço.

🚀 Funcionalidades
Cadastro e login de usuários com validação de dados

Autenticação segura

Criação e gerenciamento de agendamentos de faxina

Comunicação entre front-end e back-end via API REST

Persistência de dados em banco de dados

🧠 Tecnologias Utilizadas
Front-end: React

Back-end: Node.js + Express

Banco de dados: MongoDB

Integração: API RESTful para comunicação entre as camadas

🔗 Arquitetura
O sistema segue uma estrutura em camadas:

O front-end consome a API para exibir e enviar dados

O back-end processa regras de negócio e autenticação

O banco de dados garante a persistência e integridade das informações

💡 Objetivo
Oferecer uma solução prática e digital para organização de serviços de limpeza, centralizando cadastro de usuários e agendamentos em uma única plataforma confiável.
